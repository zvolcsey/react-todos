# Full-Stack Todo Web App

A simple full-stack todo web application with React, TypeScript and Express.
Add, complete, and delete todos with localStorage persistence.

## Features

- Delete todos
- Data persists in localStorage

- Vite
- CSS Modules

## Run Locally

```bash
  cd client
  npm install
  npm run dev

  cd ../server
  npm install
  npm run dev
```
